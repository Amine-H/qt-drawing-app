• Commandes
    • ajout
      • ligne X1 Y1 X2 Y2 : ajout un ligne dont cordinaux P1(X1,Y1) P2(X2,Y2)
      • rect X1 Y1 X2 Y2 : ajout un rectangle dont les cordinaux du premiere point a gauche est (X1,Y1) et les cordinaux du 3eme point est (X2,Y2)
      • cercle X Y R : ajout une cercle dont les cordinaux du centre sont (X,Y) et le rayon est R
      • polygon X1 Y1 X2 Y2 ... Xn Yn : ajout un polygon dont chaque deux cordonné sucssifs seront connecté (X1,Y1) est connecté avec (X2,Y2).. jusqu'a la fin don't la point final sera connecté avec la premiere point
    • set
      • color r g b : modifie la couleur selon leur valuers RGB
      • color nom : modifie la couleur selon le nom , noms disponibles (black,blue,green,red,yellow)
    • effacer
      • efface l'ecrant
    • clear
      • efface la console
